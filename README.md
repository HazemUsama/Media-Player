Simple Media Player using Python and PyQt5.

## Requirements
- tkinter
- PyGame

## Installation
- Install tkinter
- Install PyGame
- Clone this repository
- Run `python main.py`

## Features
- Play/Pause
- Stop
- Next/Previous
- Shuffle
- Repeat
- Volume Control
- Playlist
- Drag and Drop
- Fullscreen
- Open File
- Open Folder
- Slow Motion
- Fast Forward
- Mute
