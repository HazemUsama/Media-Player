Simple Media Player using Python and PyQt5.

## Requirements
- Python 3.6+
- PyQt5
- PyQt5-tools
- PyGame

## Installation
- Install Python 3.6 or higher
- Install PyQt5 and PyQt5-tools
- Install PyGame
- Clone this repository
- Run `python main.py`

## Features
- Play/Pause
- Stop
- Next/Previous
- Shuffle
- Repeat
- Volume Control
- Playlist
- Drag and Drop
- Fullscreen
- Open File
- Open Folder
- Slow Motion
- Fast Forward
- Mute
